The uses of Metadata

Metadata is simply put information about data. A simple example to illustrate what metadata is, is an envelope for a posted letter. Everything on the outside of the envelope, the address, the stamp etc. is the metadata and the data that it is providing information about is the letter inside the

Similarly semantic metadata is used in computer science to provide computers with the right information to be able to process data appropriately. Making it easier for computers to process, manage and use data. This helps give the computer some knowledge of what the data is and how it should be used.

This brings us to metadata in terms of the web. HTML tags can provide metadata to web browsers and search engines that help them process the webpage's data and display it to the right user depending on that user's needs. HTML tags also tell the web browser how the data on the webpage should be displayed to the user, as well as some knowledge onto what the information is that is actually on the webpage. HTML acts as structured referential data helping web browsers and search engines classify, find and use data in a way that is accurate and appropriate.
